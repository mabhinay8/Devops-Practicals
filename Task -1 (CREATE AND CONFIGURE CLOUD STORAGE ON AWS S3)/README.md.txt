# AWS IAM User Setup and S3 Access Guide

This guide explains how to create an IAM user with AWS Management Console access, configure login credentials, and grant permissions to use Amazon S3.

---

## Steps to Create IAM User

1. **Create User**
   - Navigate to **IAM** in the AWS Management Console.
   - Click **Users** → **Add user**.
   - Enter the username: `test-user-001`.
   - Select the checkbox: **Provide user access to the AWS Management Console**.

2. **Set Password**
   - Choose **Autogenerate password**.
   - Ensure the option **User must create a new password at next sign-in** is selected.
   - Click **Next**.

3. **Permissions**
   - For now, proceed with **default settings**.
   - Click **Next**.

4. **Create User**
   - Click **Create user**.
   - A `.csv` file will be generated containing:
     - Username
     - Password
     - Console login URL

---

## Login as Test User

1. Use the credentials from the `.csv` file to log in as `test-user-001`.
2. On first login, the user will be prompted to **change the password**.

---

## Assign Permissions

1. Log back in as the **root user**.
2. Go to **IAM → Users → test-user-001**.
3. Attach the policy: **AmazonS3FullAccess**.
   - This grants full access to Amazon S3 services.

---

## Access S3 as Test User

1. Log in as `test-user-001`.
2. In the AWS Management Console, search for **S3** in the search bar.
3. Perform the following actions:
   - Create a new S3 bucket.
   - Upload files into the bucket.

---

## Summary

- Created IAM user `test-user-001` with console access.
- Configured autogenerated password with mandatory reset.
- Granted **AmazonS3FullAccess** permissions.
- Verified login and tested S3 bucket creation and file upload.

---

## Notes

- Always download and securely store the `.csv` file containing login credentials.
- Avoid sharing root account credentials; use IAM users for daily operations.
- Apply the principle of least privilege when assigning permissions.
